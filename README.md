## Run Locally

Clone the project:

```bash
git clone https://github.com/ShivaJ007/shiva-portfolio.git
```

Go to project folder:

```bash
cd shiva-portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open in browser:

```text
http://localhost:3000
```

## Deployment

This project is deployed on [Vercel](https://vercel.com/).
Every push to the `main` branch automatically triggers a new deployment.
