# Shiva Portfolio

Personal portfolio website built with Next.js to showcase my projects, skills, and work.

## Live Demo

`https://shiva-portfolio-alpha.vercel.app/`

## Features

* Responsive portfolio design
* Projects showcase
* About section
* Resume section
* Contact section
* Fast deployment with Vercel

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* Vercel

## Run Locally

Clone the project:

```bash
git clone https://github.com/your-username/shiva-portfolio.git
```

Go to project folder:

```bash
cd shiva-portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## Deployment

This project is deployed on Vercel.

Every push to the `main` branch automatically triggers a new deployment.
